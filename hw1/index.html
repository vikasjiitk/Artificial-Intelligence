Description

The data of distance travelled by dropping ball for different interval of time is given. We have to come with a suitable model using Curve Fitting to correctly represent the data using training set. For different models, the error is estimated on training set, validation set and test set. Based on the result of the errors a suitable model is chosen. Also we have to check what difference it make to analyze models using a small Training Set and Big Training Set. Homework description link.

  • Curve Fitting Analysis on Small Training Data Set

  • Curve Fitting Analysis on Big Training Data Set

Final Result

From both small training data set and large training data set, the optimal model for the given data is Quadratic.

Difference between Validation set and Test set

The validation set is the data set used to validate the different models you have at start and finding best model from the validation set results. Where as, the test data is used to test the accuracy of your chosen model and find out the error of your model.
